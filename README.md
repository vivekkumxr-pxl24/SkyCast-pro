# 🌤️ SkyCast Pro - Weather Dashboard

SkyCast Pro is a modern, responsive, and immersive weather web application. It provides users with real-time weather updates, detailed atmospheric statistics, and a 5-day outlook using a sleek "Glassmorphism" UI design.

## ✨ Features
- **Real-time Weather:** Get instant temperature, humidity, and wind speed for any city.
- **Dynamic Backgrounds:** The entire page theme changes according to the current weather (e.g., Rainy, Clear, Cloudy).
- **5-Day Forecast:** Detailed daily outlook with high-resolution weather icons.
- **AQI Monitoring:** Real-time Air Quality Index (Good, Fair, Moderate, etc.).
- **Sunrise & Sunset:** Precise timings for daylight tracking.
- **Geolocation Support:** One-click weather detection for your current location.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties, Flexbox, Grid, and Glassmorphism effects.
- **JavaScript (ES6):** Fetch API, Async/Await, and Geolocation API.
- **API:** [OpenWeatherMap API](https://openweathermap.org/).
